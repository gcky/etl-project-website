Ready for Tungsten?

Follow these easy steps to get started.

Step 1: Install Vagrant

Tungsten requires Vagrant so go on and install it on your machine.

Download Vagrant

Step 2: Download Tungsten

Download the Tungsten vagrant file and put it in a folder.

Download Tungsten

Step 3: Set-up VM

Open terminal/console, cd to the folder you created and do vagrant up.

You're ready to go!
