Ready for Tungsten?

Follow these easy steps to get started.

Step 1: Install VirtualBox & Vagrant

Tungsten requires VirtualBox and Vagrant so go on and install them onto your machine.

Get VirtualBox Get Vagrant

Step 2: Download Tungsten

Create a directory for Tungsten, download the Tungsten Vagrantfile and put it in the directory.

Download Vagrantfile

Step 3: Set-up VM

Open terminal/command prompt, set current directory to the one you just created and do vagrant up.
Have a nice cup of tea while Tungsten is being set-up.

You're ready to go!

Open your browser and go to localhost:5000, upload a dataset and start analysing!
Read the user guide for detailed instructions on how to use Tungsten.
